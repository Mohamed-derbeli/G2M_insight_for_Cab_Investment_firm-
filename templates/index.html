Model Description


This web application is designed to help users of Pink and Yellow Cab companies by providing information regarding the prices. The aim of this project is to predict the price charged by each company based on certain information. The application requires several input data that should be entered by the user. Finally, the predict bottom will generate the price charged based on the user data.
